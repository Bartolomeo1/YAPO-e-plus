On windows and linux, chromedriver.exe or chromedriver binary go in this folder. Remember to set it executable on Linux (chmod 777 chromedriver)

Download the latest release here: https://chromedriver.chromium.org/downloads

Currently, the latest release is 84.0.4147.30, which can be found here:

Windows:  https://chromedriver.storage.googleapis.com/84.0.4147.30/chromedriver_win32.zip
Linux:    https://chromedriver.storage.googleapis.com/84.0.4147.30/chromedriver_linux64.zip

Or navigate the Chromedriver site here: https://chromedriver.chromium.org/


Chromedriver is necessary to download additional actor images, which will be done thru this webdriver to avoid Google's limitation of 1000 image searches.
